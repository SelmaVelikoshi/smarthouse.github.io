Project Overview

The IoT Smart House Door Lock System is a cutting-edge project designed to enhance home security through the integration of various sensors and control mechanisms. The system employs an RFID sensor to authenticate users, an ultrasonic sensor to detect the presence of individuals, and two servo motors to control the opening of the main gate and the house door. Additionally, the system utilizes an ESP8266 module for logging access attempts and providing remote control via a web interface.

When a user approaches the system, they must scan their RFID tag. The RFID reader checks the tag against a list of authorized tags. If the tag is authorized, the servo motor connected to the main gate activates, allowing entry. If the tag is not recognized, the system logs the attempt and sends an alert, and the ultrasonic sensor remains deactivated to prevent unauthorized access.

Once the main gate is opened, the ultrasonic sensor comes into play. It detects if a person is within a 2-meter range of the door. If presence is detected, the second servo motor is triggered to open the house door, allowing the user to enter. This two-step verification process ensures that only authorized individuals can access the house, even if they manage to get through the main gate.

The ESP8266 module is a crucial component of the system, responsible for logging the date and time of each access attempt. This data can be monitored remotely through a web interface, providing homeowners with real-time updates on the status of their entry points. Furthermore, the web interface allows the owner to manually control the door, granting access to visitors who do not have an RFID tag.

In the event of an internet failure or RFID system malfunction, a manual override feature is available. A keypad is installed to allow entry using a specific code, ensuring that the security system remains functional under various circumstances.

Features

  • RFID authentication
  • Ultrasonic sensor for presence detection
  • Servo motors for gate and door control
  • ESP8266 for logging and remote access
  • Manual override with keypad
